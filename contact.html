Technical Support

If you've encountered a bug or a parsing error with your resume:

  1. Click the "Download Debug Log" button in the footer of the main page.
  2. Attach the generated JSON file to your email.
  3. Send it to hello@getatsready.com.

Note: The debug log contains metadata about the parsing process but does NOT include your full resume text or sensitive personal information.
