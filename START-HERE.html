🧪 Quick Testing Guide for ATS Checker

Option 1: Test in Browser (Easiest)

Step 1: Open index.html in Chrome
Step 2: Click "Try with Example Resume"
Step 3: Click "Analyze Resume"
Step 4: Check if score appears (should be 75-85%)
Step 5: Download PDF report
Step 6: Press F12 → Click phone icon → Test mobile view

Option 2: Test on Real Phone (Best)

Step 1: Open PowerShell and run this command:
cd "C:\Users\Vinod\Desktop\Website ideas\ATS-Checker"
npx http-server -p 8000
Step 2: Find your computer's IP address:
ipconfig
Look for "IPv4 Address" under "Wireless LAN adapter Wi-Fi"
Example: 192.168.1.100
Step 3: On your phone (connected to SAME Wi-Fi):
  • Open browser
  • Type: http://192.168.1.100:8000 (use YOUR IP)
  • Test uploading resume and analyzing

Option 3: Deploy to Internet (Recommended)

Step 1: Install Vercel CLI:
npm install -g vercel
Step 2: Deploy:
cd "C:\Users\Vinod\Desktop\Website ideas\ATS-Checker"
vercel
Follow the prompts (login with Google/GitHub)
Step 3: You'll get a URL like:
https://ats-checker-abc123.vercel.app
Open this URL on ANY device (phone, tablet, laptop) to test!
⚡ This is FREE! Vercel won't charge you anything. Your site will be live on the internet in 2 minutes.

✅ What to Test

  1. Try Example: Click "Try with Example Resume" button
  2. Analyze: Click "Analyze Resume" - should show score 75-85%
  3. Keywords: Check found/missing keywords make sense
  4. PDF: Download PDF report - should generate correctly
  5. Upload: Try uploading your own resume (PDF or DOCX)
  6. Mobile: Test on phone - everything should be readable
  7. Navigation: Click About, Privacy, Contact - all should load
  8. Cookie Banner: Should appear at bottom, dismiss when clicked

🐛 Common Issues

  • File upload not working: Check if you're using Chrome/Firefox (Safari can be buggy)
  • No score appearing: Open DevTools (F12), check Console for errors
  • PDF not downloading: Make sure pop-ups aren't blocked
  • Can't access from phone: Ensure phone and computer are on SAME Wi-Fi

💡 Pro Tips

  • Test with at least 3 different resumes (Software Engineer, Marketing, Data Analyst)
  • Test edge cases: very short resume, very long resume, wrong file type
  • Ask a friend to test - fresh eyes catch more bugs!
  • Compare results with Jobscan.com to validate accuracy

🚀 Ready to Deploy?

Once you've tested everything and it works:

  1. Run: vercel --prod (deploy to production)
  2. Buy domain from Namecheap/Hostinger (₹800)
  3. Connect domain in Vercel dashboard
  4. Apply for Google AdSense
  5. Start promoting!

Total time: 1-2 hours. Total cost: ₹800.
